
# 🚀 Day 05 - Fibonacci Generator

This Java program generates the Fibonacci series up to a specified number of terms.

---

## 📌 How to Run

1. Open terminal or command prompt.
2. Navigate to the project directory.
3. Compile the Java file:
    ```bash
    javac Main.java
    java Main
