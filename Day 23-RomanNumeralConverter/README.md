## â–¶ How It Works

Roman numerals follow specific patterns:
- `I = 1`
- `V = 5`
- `X = 10`
- `L = 50`
- `C = 100`
- `D = 500`
- `M = 1000`

Example:

- **4 = IV** (5 - 1)
- **9 = IX** (10 - 1)
- **58 = LVIII** (50 + 5 + 3)
- **1994 = MCMXCIV**
